===============================================================
HSP-R — Highly Sensitive Person Scale - Revised
Veil-Inspired Tarot Preview (Local Only)
===============================================================

STATUS: This is a LOCAL PREVIEW of a tarot-inspired visual theme.
It has NOT been approved for production and has NOT been pushed
to any repository. For review purposes only.

DESIGN INSPIRATION:
The visual language draws from the "Veil" Dribbble concept:
minimalist, mystical, serene aesthetics. No images, logos,
card artwork, or assets from the original Veil author have been
copied. Only the visual language was studied: dark backgrounds,
thin orbital lines, deep purple tones, softly glowing cards,
and subtle motion. All graphics are pure CSS.

How to run:
-----------
Simply open the file "index.html" in any modern web browser
(e.g. Chrome, Firefox, Edge, Safari). No installation, server,
or internet connection is required. All fonts are self-hosted
locally under assets/fonts/.

Files included:
---------------
- index.html  : Main page (open this file)
- styles.css  : Veil-inspired styles and layout
- app.js      : Application logic and scoring (data/formulas unchanged)
- README.txt  : This file

Features:
---------
- 18 questions from the HSP-R (English original)
- 3 languages: English, 日本語 (Japanese), Tiếng Việt (Vietnamese)
- Likert scale 1-7 (1 = Not at all, 7 = Extremely)
- Questions may be left unanswered (blank allowed)
- Overall total score, overall average, answered count
- 6 subscale groups with per-group totals and averages
- Threshold > 5 for high sensitivity tendency
- All processing is local in your browser; no data is collected or sent
- No CDN, no API, no external resources

Visual design & Motion system:
------------------------------
- Dark ritual-space hero with CSS orbital rings, breathing opacity,
  and small celestial points (pure CSS, no SVG or copied artwork)
- Slowly drifting purple/gold aurora glows in the page background
- Hero title with restrained purple-to-ivory shimmer and soft entrance
- Animated light-strip progress bar (background-position shimmer)
- Question cards gently revealed on scroll via IntersectionObserver
  with immediate fallback if JS or observer is unavailable
- Warm edge glow and slight lift on card hover
- Selected 1-7 circles emit a purple/gold pulse and halo
- Keyboard focus (gold outline) is clearly distinct from selection
- Subtle light sweep on the primary action button
- Elegant reveal animation on the result card
- Faint pointer-following aura on fine-pointer desktop devices only;
  never interferes with clicking and never runs on touch devices
- Cormorant Garamond for display headings and tarot accents
- Manrope for body text, questions, and buttons
- Color palette: deep black (#020207), dark surface (#0b0a18),
  purple (#625FDE), muted gold (#AF9D89), light text (#ECE3E0)
- Question text slightly brighter for readability
- Glassmorphism pill language selector
- Tarot-style question cards with thin gold borders
- Responsive for mobile and desktop
- Accessible: focus-visible, good contrast, semantic fieldset/legend
- Respects prefers-reduced-motion (disables all nonessential animation)
