===============================================================
HSP-R — Highly Sensitive Person Scale - Revised
===============================================================

How to run:
-----------
Simply open the file "index.html" in any modern web browser
(e.g. Chrome, Firefox, Edge, Safari). No installation, server,
or internet connection is required.

Files included:
---------------
- index.html  : Main page (open this file)
- styles.css  : Styles and layout
- app.js      : Application logic and scoring
- README.txt  : This file

Features:
---------
- 18 questions from the HSP-R (English original)
- 3 languages: English, 日本語 (Japanese), Tiếng Việt (Vietnamese)
- Likert scale 1-7 (1 = Not at all, 7 = Extremely)
- Questions may be left unanswered (blank allowed)
- Overall total score, overall average, answered count
- 6 subscale groups with per-group totals and averages
- Responsive design for mobile and desktop
- Accessible: keyboard navigation, focus styles, aria-labels
- All processing is local in your browser; no data is collected or sent
